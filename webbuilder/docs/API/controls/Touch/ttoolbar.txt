<p class="wb_h1">TToolbar</p>
Ext.Toolbars are most commonly used as docked items as within a Ext.Container. WebBuilder encapsulate Sencha Touch Ext.Toolbar to this control.<br><br>
<p class="wb_h2">1. Properties</p>
<hr>
<b>autoCreate</b><br>
Indicates whether to create the object instance or config object.<br>
<hr>
<b>type</b><br>
Specify the toolbar type, options are toolbar or titlebar.<br>
<hr>
<b>tagProperties</b><br>
Any text to be appended to the object's config options directly.<br><br>
<p class="wb_h2">2. Events</p>
<hr>
<b>tagEvents</b><br>
Any text to be appended to the object's listeners directly.<br><br>
<p class="wb_h2">Others please refer to Sencha Touch docs Ext.Toolbar</p>
